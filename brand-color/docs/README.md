# dgdgdgdg Brand Color

**dgdgdgdg** のブランドカラーパレット

![dgdgdgdg Brand Color](./dgdgdgdg-brand-color.png)

## プロジェクト概要

「dgdgdgdg」のアイデンティティを形成する 729 色のブランドカラーパレットを表示するインタラクティブ Web アプリケーション。16 進数表記の `0`, `4`, `d` のみを組み合わせた色彩体系を視覚化し、色の探索を可能にします。

## カラーパレット仕様

### 色の定義

16 進数 `0`, `4`, `d` を RGB 各チャンネルの 2 桁に配置することで色を定義：

- 使用可能な 2 桁の組み合わせ: `00`, `04`, `0d`, `40`, `44`, `4d`, `d0`, `d4`, `dd`（9 通り）
- 全色数: 9 × 9 × 9 = **729 色**
- メインカラー: `#00d4d4`（開始色として使用）

### 色の並び順

色空間における最近傍法（Nearest Neighbor）により、視覚的に連続したグラデーションで配置：

1. 開始色 `#00d4d4` を起点とする
2. RGB ユークリッド距離 `√((r1-r2)² + (g1-g2)² + (b1-b2)²)` が最小の未配置色を次に選択
3. 全 729 色が配置されるまで繰り返す
