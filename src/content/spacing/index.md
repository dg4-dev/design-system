# dgdgdgdg Spacing

**dgdgdgdg** のスペーシング（余白）体系

## プロジェクト概要

余白の基準値とスケールを定義し、レイアウト・コンポーネント間の一貫した間隔を実現します。

## スペーシングスケール

基準: 0.25rem

| トークン | 値      | 用途例             |
| -------- | ------- | ------------------ |
| 0        | 0rem    | リセット           |
| 1        | 0.25rem | 極小（アイコン内） |
| 2        | 0.5rem  | コンポーネント内   |
| 3        | 0.75rem | 狭い余白           |
| 4        | 1rem    | 標準               |
| 5        | 1.25rem | やや広い           |
| 6        | 1.5rem  | セクション内       |
| 8        | 2rem    | セクション間       |
| 12       | 3rem    | ブロック間         |
| 16       | 4rem    | ページ区切り       |
